## Проект "Анализатор качества репозиториев"

Задеплоен на [render](https://analizator-kachestva-repozitoriev.onrender.com/)

## Local installation
```
make install-without-production
```
## Start dev-server
```
make dev-start
```
## Start linters
```
make lint
```
## Start tests
```
make test
```
